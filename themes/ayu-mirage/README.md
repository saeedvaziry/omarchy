# Ayu Mirage Omarchy Theme

This is the mirage variant of [Ayu](https://github.com/dempfi/ayu) for [Omarchy](https://github.com/dempfi/omarchy).
<div align="center">
    <img src="theme.png" alt="Screenshot of Ayu Mirage Omarchy Theme">
</div>

## Installation

To install this theme run the following command:

```bash
omarchy-theme-install https://github.com/fdidron/omarchy-ayu-mirage-theme
```

in your terminal.

Light version of this theme is available [here](https://github.com/fdidron/omarchy-ayu-light-theme).
